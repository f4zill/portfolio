Gym Data Management System

Full-stack app to manage gym data.

A full-stack web application designed to streamline and automate core gym operations, including the management of members, trainers, equipment, workout logs, and payments. The system enables efficient, real-time CRUD operations and supports seamless handling of over 500 records through a scalable and modular architecture. Key Technical Highlights: Developed a full-stack data management platform with integrated CRUD functionality across six primary entities: members, trainers, equipment, workouts, payments, and subscriptions. Utilized MongoDB as a flexible, document-based NoSQL database to support scalable and high-performance storage and retrieval operations. Implemented backend functionality using Flask, with Jinja2 templating to dynamically render HTML and support real-time user interactions. Designed clean, user-friendly web interfaces using HTML/CSS, facilitating accurate and intuitive data input and navigation. Structured the application modularly to ensure maintainability, scalability, and ease of future enhancements such as authentication, analytics, or reporting dashboards. Integrated clear routing and form validation through Flask routes and blueprint organization, ensuring consistent and secure data flow between client and server.

Tech: Flask, Python, MongoDb, Render

Chest X-ray Pneumonia Detection System

AI-powered Diagnostic Tool for Pneumonia Detection from Medical Imaging

Developed an AI-driven diagnostic system using TensorFlow, Keras, and Flask to assist healthcare professionals in detecting pneumonia from chest X-ray scans. The model, trained on a dataset of 5,000+ labeled medical images, achieves 92%+ accuracy and classifies scans as Normal or Pneumonia. Implemented Grad-CAM visualizations to highlight critical lung regions, improving interpretability for clinicians. Integrated the model into a fully responsive web interface where users can upload scans, view predictions instantly, and interact with a modern, animated UI built with HTML/CSS, Tailwind, and JavaScript. Designed for scalable deployment and optimized to maintain smooth user experience without page reloads.

Tech:Python, TensorFlow, Keras, Flask, HTML/CSS, Grad-CAM, Jinja2, Bootstrap
